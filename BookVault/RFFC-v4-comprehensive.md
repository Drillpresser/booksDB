# Robinson Fiction-Forward Classification (RFFC)
## Comprehensive Edition — v4.0

*A fiction-weighted classification system designed to give every book in a general collection exactly one home.*

---

# PART 1 — SYSTEM RULES

## The Four Levels

| Level | Name | Example | What it decides |
|---|---|---|---|
| 1 | **Class** | `500` | The genre or subject family (the reader's promise) |
| 2 | **Division** | `.20` | The shelf neighborhood (subgenre or subject area) |
| 3 | **Section** | `.03` | The precise slot — the spine number |
| 4 | **Suffix + Tags** | `–a , scifi` | Physical form, audience, and secondary content |

**Notation:** `CLASS.DIVISION.SECTION – suffix , tags`
Example: `500.20.03 –a , scifi` → Horror → Monster → Zombies, anthology, with sci-fi crossover elements.

## Design Principles (v4.0)

1. **One Home Principle.** Every concept appears in exactly one place in the schedules. When two classes could plausibly claim a book, the Boundary Rulings (Part 3) decide — not case-by-case judgment. Consistency beats cleverness.
2. **Full number space.** Sections run `.01`–`.99` and divisions `.10`–`.90` with no reserved ranges. The former `.21+` local-extension rule is retired.
3. **Granularity on demand.** When a category regularly holds books that feel unlike their neighbors, it earns its own section. When in doubt, a new classification was created rather than a judgment call left open.
4. **This document is the schema.** Extensions are made by taking the next open number in the correct division and recording it here. A number in use on a spine is never reassigned.

## The Priority Ladder (fiction)

When classifying fiction, decide in this order — each rung only refines, never overrides, the rung above:

1. **Promise → Class.** What does the book promise the reader? (Fear → 500. The couple → 400. The puzzle/threat → 300. The speculative world → 200. The prose/portrait → 100.)
2. **Engine → Division.** What machinery delivers the promise? (Ghosts vs. monsters vs. dread; detective vs. thriller vs. noir.)
3. **Setting/subject → Section.** The specific flavor within the engine.
4. **Everything else → Suffix + tags.** Audience and format never move a book between classes; secondary genres become tags.

## The Priority Ladder (nonfiction)

1. **Reading purpose → Class.** Understanding the past → 600. Understanding the natural/technical world → 700. Belief, myth, and meaning → 800. Making, performing, appreciating → 900. Looking things up or running a life → 000.
2. **Subject → Division.** 3. **Specific topic → Section.** 4. **Form/tags** as with fiction.

## Form & Audience Suffixes (Level 4a — use at most one)

| Suffix | Meaning |
|---|---|
| `–a` | Multi-author anthology / collection |
| `–s` | Single-author short story collection |
| `–g` | Graphic novel, comic, or manga format |
| `–j` | Juvenile / middle grade audience |
| `–y` | Young adult audience |
| `–o` | Oversized / coffee-table format |
| `–r` | Reference format (consulted, not read through) |
| `–x` | Local or personal association (signed, family, local author) |

## Cross-Genre Tags (Level 4b)

Lowercase words after a comma, for genuine **secondary** content only. Never tag a book with its own class family. Core vocabulary (extend freely):

`literary` · `scifi` · `fantasy` · `mystery` · `thriller` · `romance` · `horror` · `history` · `science` · `religion` · `myth` · `poetry` · `humor` · `war` · `sports` · `gaming` · `western` · `dystopia` · `sweet` · `steamy`

(`sweet` and `steamy` are optional heat-level facets for Romance outside divisions 400.60/400.70.)

## The 60-Second Flow

1. Fiction (100–500), nonfiction (000, 600–800), or arts/making/play (900)?
2. Climb the Priority Ladder to a class, division, section.
3. Check the **Boundary Rulings** if the book sits near a known border.
4. Add one suffix if the form or audience warrants; add tags for real secondary genres.
5. When two sections both fit, take the one earlier in its division (more general wins).

# PART 3 — BOUNDARY RULINGS

These are binding. When a book sits on one of these borders, the ruling decides — no judgment call required.

| # | Border | Ruling |
|---|---|---|
| R1 | True crime (nonfiction) | Always **300.80**, never 600. Crime readers browse fiction and fact together. Tag `history` if the case is historical. |
| R2 | Musician biographies | Always **900.20.02**, never 600.60. Music readers expect them with music. All other artist bios → 600.60.03. |
| R3 | Medical memoirs | Physician/patient narratives about medicine → **700.80.06**. Life stories where medicine is incidental → 600.60.08. Test: is the medicine the point? |
| R4 | Romance heat extremes | Explicitly sweet/inspirational → **400.60**; erotic/dark → **400.70** — these outrank subgenre. An erotic Regency is 400.70.02, not 400.20.01. Mid-heat books class by subgenre with optional `sweet`/`steamy` tags. |
| R5 | Romantasy vs. fantasy | If removing the romance collapses the book → **400.30**. If removing it leaves a working fantasy novel → **200**, tag `romance`. |
| R6 | War history vs. military history | Combat, campaigns, strategy, units → **600.50**. Home fronts, politics, societies at war → the era or region division. |
| R7 | Serial killers | Investigation payoff → **300**. Dread payoff (killer's shadow, victim POV) → **500.70.03**. Nonfiction → 300.80.02. |
| R8 | Myth: retelling vs. subject | Novelized retellings → **200.30.03** (or 100 if literary, tag `myth`). The myths themselves, studies, and source texts → **800.50**. |
| R9 | Dystopia vs. post-apocalyptic | Oppressive functioning system → **200.20.01**. Collapsed world → **200.20.02**. Collapse *with* fear payoff → 500.80.03. |
| R10 | Gothic vs. historical fiction | Atmosphere of dread and the uncanny → **500.50**. Past-as-portrait without dread → 100.30. |
| R11 | Paranormal cozy vs. urban fantasy | Mystery-of-the-week engine → **300.20.06**. Worldbuilding/adventure engine → 200.40.01. |
| R12 | Self-help vs. philosophy of living | Rooted in a wisdom tradition (Stoicism, Buddhism) → **800**. Modern secular technique → **000.60**. |
| R13 | Gardening | Plant science → **700.30.07**; nature/field guides → 700.40.03; growing things on purpose → **900.50.07**. |
| R14 | Cooking | Recipes & technique → **900.50.06**. Food history & culture → 600.70.02. Nutrition science → 700.80.04. |
| R15 | Disasters | Narrative accounts → **600.70.05**. The science of the hazard → 700.50.06. Disaster thrillers → 300.40.06. Apocalyptic fiction → 200.20.02 or 500.80. |
| R16 | Translated works | Contemporary literary in translation → **100.80**. Translated classics → 100.20 by era. Translated genre fiction → its genre class (300.10.06 exists for crime). |
| R17 | Anthologies | Genre anthologies stay in their genre's .90 division with `–a`. Only genre-less literary anthologies use 100.90. |
| R18 | Legend cycles (Arthur, Robin Hood) | Fiction → **200.50.05**. Scholarship & source texts → 800.60.05. |
| R19 | Series that drift | The whole series classes where **book one** lives. Consistency beats precision. |
| R20 | Verse novels | Narrative dominates → the story's genre class, tag `poetry`. Language dominates → 900.10.04. |

# PART 5 — MAINTENANCE & EXTENSION

**Adding a section.** Find the correct division, take the next unused number, add it to this document with a one-line scope note if it borders anything. Never leave a new section undocumented.

**Adding a division.** Only when a theme outgrows section status (rough test: 15+ books that would benefit from their own neighborhood). Take the next open tens slot in the class.

**Never renumber in-use numbers.** A number on a spine is permanent. If a section was a mistake, close it to new books, note the successor, and migrate at leisure.

**When a new border appears,** write a ruling. Add it to Part 3 with the next R-number. The rulings table is what keeps two people classifying the same book the same way.

**Version this document.** Date each change. The schema doc — not the shelves, not memory — is the single source of truth.
